Instructions

  • Click anywhere on the above canvas to select a random starting point
  • Click start to see the simulation play
  • Click pause to restart the entire process
  • When paused, the triangle points can be dragged to different positions

How does it work?

  1. You select an inital random point [P1]
  2. Out of the 3 points of triangle, a random point is selected [P2]
  3. We calculate the mid point between the random point and the triangle point [P1 & P2]
  4. Assume this new point to be P1
  5. Repeat the step 2.
